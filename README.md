# F1 & BI
### Dashboard com os dados das temporadas de 1950-2023 da Formula1.

## Origem dos dados
O dataset utilizado nesse projeto está disponível no [Kaggle](https://www.kaggle.com/datasets/rohanrao/formula-1-world-championship-1950-2020) graças ao [Vopani](https://www.kaggle.com/rohanrao).
